## 🚀 Projetos em destaque

### Workflow Approval Engine
Plataforma full stack para criação e execução de fluxos de aprovação corporativos, com recursos como:

- designer visual de fluxos
- versionamento
- trilha de auditoria
- controle de acesso por perfil
- dashboards com KPIs e SLA

🔗 Repositório: [workflow-final](https://github.com/gustavohenriqq/workflow-final)

### CodeSentinel
Ferramenta voltada para auditoria e análise de código, com foco em organização das análises, acompanhamento de resultados e apoio à identificação de pontos de atenção no desenvolvimento.

🔗 Repositório: [CodeSentinel](https://github.com/gustavohenriqq/CodeSentinel)
